# Engagement Signal
# 检测 "User 对某个话题产生了持续兴趣"

## 触发模式

- 同一话题连续聊了 ≥3 轮
- User 主动追问细节（"这个怎么做到的" "底层逻辑是什么"）
- User 说 "好好玩" "有意思" "继续"
- 回复变长、提问频率增加 → 认知投入信号

## 响应

- 自动提升该话题的 dopamine_score
- 写入热存储：[engagement] 话题 × 粘性 × 轮次
- 不打断——只在 User 自然暂停时询问要不要深潜
- 记录到经验库：什么类型的话题引发了 engagement

## 实现思路

Agent 每轮回复前检查最近 5 轮对话：
- 高频关键词聚类 → 识别核心话题
- 轮次数 ≥ 3 → 标记为 "engaged"
- 反馈格式："你对 [话题] 连续聊了 [N] 轮，要深潜吗？还是先记下来回头再说？"
