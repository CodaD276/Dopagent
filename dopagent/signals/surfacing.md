# Surfacing Signal
# 检测 "User 在反思 / 表达模糊感悟"

## 触发模式

- "其实我觉得..." "仔细想想..." "本质上是..."
- User 在自我分析——不是问问题，是在梳理思绪
- 情绪表达 + 认知加工混合（"我好像发现了一个规律"）
- 元认知——对自己思考方式的观察

## 响应

- 不打断，不急着给结论
- 把 User 的模糊表述**镜像回去**："你是说 [结构化重述] 对吗？"
- 问引导性问题，帮他把隐性感悟变成显性表述
- 如果 User 确认 → 自动写入热存储：[insight]
- 标记为高价值——这类 insight 是经验库最优质的燃料

## 实现思路

Agent 检测 User 回复中是否含：
- 推理性连接词（"本质""其实""仔细想"）
- 自我修正句式（"我之前以为...但其实..."）
- 抽象概括（"说到底...""根源是..."）

检测到后 → 静默标记 → 2 轮内找自然空隙→ 镜像复述 + 提议记录
