# Creative Profile
# 创意模式——高新奇度，低紧迫度，深度优先

## 权重分配

| 参数 | 权重 | 理由 |
|---|---|---|
| novelty | 0.40 | 创意模式由新奇驱动 |
| urgency | 0.05 | 创意没有 deadline |
| challenge_match | 0.30 | 难度要刚好——太简单无聊，太难焦虑 |
| autonomy | 0.25 | 自主性 = 创作的核心燃料 |

## 触发条件

- User 说 "我想试试..." "有没有可能..." "设计一个..."
- 当前没有紧迫的外部 deadline
- 时间充裕（晚间/周末 > 工作日碎片时间）

## 不适用信号

- 财务焦虑期（月初效应消退前 → 切到 recovery）
- 物理疲惫（Apple Health HRV 低 → 不适合深潜创意）
