# Exploration Profile
# 探索模式——纯兴趣驱动，零紧迫度，广度优先

## 权重分配

| 参数 | 权重 | 理由 |
|---|---|---|
| novelty | 0.50 | 探索就是找新东西 |
| autonomy | 0.30 | 完全自主——"感觉好好玩" |
| urgency | 0.00 | 没有 deadline，没有压力 |
| challenge_match | 0.20 | 不需要太难，有趣就行 |

## 触发条件

- User 说 "还有什么好玩的" "社区上最近有什么"
- 刚完成一个产出周期，在寻找下一个触发
- 社交场景——和朋友讨论 → 被安利 → 好奇心被点燃

## 行为模式

- Agent 主动推送高新颖度内容（不催行动，只提供选项）
- 广度优先——一次给 3-5 个方向，不是深挖一个
- 记录 User 的反应——对什么表现出兴趣 → 写入 engagement 信号
