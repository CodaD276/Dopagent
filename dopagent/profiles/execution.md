# Execution Profile
# 执行模式——高紧迫度，微步骤分解，快速闭环

## 权重分配

| 参数 | 权重 | 理由 |
|---|---|---|
| urgency | 0.40 | 执行由紧迫度驱动 |
| novelty | 0.10 | 不是找刺激的时候 |
| challenge_match | 0.25 | 难度略低于 sweet spot——确保能完成 |
| autonomy | 0.15 | 允许一定外部依赖（如 subagent 委派） |
| micro_step_fit | 0.10 | 任务能不能拆成 <30min 的微步骤 |

## 触发条件

- 有明确产出目标（"今晚要把这个做完"）
- User 说 "继续" "开干" "做吧"
- 刚从 hyperfocus 浮出水面，需要收尾

## 执行策略

- 自动拆解任务为微步骤（每步 <30min）
- 每完成一步 → 主动汇报进度（"第二步完成，还剩三步"）
- 阶段性正面反馈——进度条 = 微型多巴胺注入
