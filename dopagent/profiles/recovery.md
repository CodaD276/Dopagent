# Recovery Profile
# 低能模式——极小步骤，外部问责，身体加倍

## 权重分配

| 参数 | 权重 | 理由 |
|---|---|---|
| micro_step_fit | 0.40 | 能不能拆到 <5min |
| autonomy | 0.05 | 低能时自主性不是优势——需要外部结构 |
| urgency | 0.15 | 微弱的紧迫度帮助启动 |
| novelty | 0.10 | 不需要新奇——需要熟悉和安全感 |
| social_scaffold | 0.30 | 外部问责/身体加倍是低能时最强的触发器 |

## 触发条件

- User 说 "不想动" "没动力" "算了"
- 入神失败，尝试了 >1 个任务都无法进入 hyperfocus
- 月底财务焦虑期
- Apple Health 显示低活动量 / 低 HRV

## 执行策略

- 不催。不给宏大目标
- 给一个 <5min 的微任务（"把这条 retain 了吧，30 秒"）
- 提议身体加倍——"要不拉个朋友一起？"
- 如果做了 → 正面反馈，不强推下一个
- 如果没做 → 不追，切换到其他能入神的事（STIMULUS_SEEKING）

## 你的独特策略（来自 2026-07-13 讨论）

入神失败不硬刚 → 切到能入神的事 → 深潜 → 浮出水面 →
社交沟通/自我探索 → 显性化感悟 → 存入经验库
